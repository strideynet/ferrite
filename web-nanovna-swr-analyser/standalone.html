NanoVNA SWR Analyser

Web-based S-Parameter Analysis for Amateur Radio

About This Component

This is a standalone web component that analyzes S-parameter files (S1P/S2P) from NanoVNA and similar RF analyzers. It automatically detects amateur radio bands within your sweep data and generates SWR (Standing Wave Ratio) graphs for each band.

How to Embed

This component can be easily embedded into any static website. Here are two methods:

Method 1: Auto-mount with data attributes

<!-- Include the built library --> <script src="nanovna-swr-analyser.js"></script> <!-- Create a div with auto-mount attribute --> <div id="swr-analyser" data-nanovna-swr-auto-mount data-show-all-bands="false" data-chart-height="350"> </div>

Method 2: Manual mounting with JavaScript

<!-- Include the built library --> <script src="nanovna-swr-analyser.js"></script> <!-- Create a container div --> <div id="my-analyser"></div> <!-- Mount the component --> <script> window.NanoVNASWRAnalyser.mount('my-analyser', { showAllBands: false, chartHeight: 350, onDataLoaded: function(data) { console.log('Bands found:', data); } }); </script>

Supported Amateur Bands

The component automatically detects the following amateur radio bands:

HF Bands

160m, 80m, 40m, 30m, 20m, 17m, 15m, 12m, 10m

VHF/UHF Bands

6m (50 MHz), 2m (144 MHz), 70cm (420-450 MHz)
